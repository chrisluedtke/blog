How to Embed Python from a Jupyter Notebook

The super easy way

February 10, 2018

Let's not get too technical

Unless I minimize barriers to the sharing process, this blog will remain a barren wasteland. I want the simplest option among the many that span a large range of prerequisite technical know-how. After all, I am not a CSS of HTML wiz. Thankfully GitHub pages like this one only serve static content, so I just need the code to appear correctly formatted.

The Process

  1. In a Jupyter Notebook, choose File, Download As, HTML
  2. Place the HTML file in your GitHub repo or webpage environment in a new folder
  3. In the HTML of your blog entry, add the following iframe, and experiment to set an appropriate height:

<iframe src="codeshare/Blog_sample.html" width="100%" height="200" scrolling="no" frameBorder="0"></iframe>
